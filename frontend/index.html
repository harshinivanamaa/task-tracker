📘 Welcome to Task Tracker

Task Tracker is a modern productivity web application designed to help users organize daily activities, manage deadlines, and track task progress efficiently.

🚀 Features

  • ➕ Add new tasks with title, subject, deadline and priority
  • ⏳ Manage all pending tasks separately
  • ✅ Mark tasks as completed instantly
  • 🗑 Move unwanted tasks to Trash
  • ♻ Restore deleted tasks anytime
  • 🌙 Dark mode support
  • 📱 Responsive design for desktop and mobile

🛠 How To Use

  • 1️⃣ Enter task details in the form above
  • 2️⃣ Click on Add Task to save your task
  • 3️⃣ Use the Pending page to manage active tasks
  • 4️⃣ Mark tasks as completed after finishing them
  • 5️⃣ Deleted tasks can be restored from Trash
